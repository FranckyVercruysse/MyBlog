Create sidebar widgets

The Sidebar contains stack of widgets like subscribe to my feed, latest posts, categories, tags and recent comments. You may even want to add some more.
All these widgets take 33% of the screen in desktops and 100% in mobile.




Display the post categories in a widget

The following are the tasks we have to execute.
  1. Create a repository method that returns all the categories
  2. Create an action to return posts belongs a particular category
  3. Create a view model
  4. SidebarViewComponent

Create a repository method that returns all the categories

Define and implement the method Categories that returns all the categories from database. 
namespace MyBlog.Repositories
{
    public interface IBlogRepository
    {
        ...

        IList<Category> Categories();
    }
}


The implementation is quite simple,

namespace MyBlog.Repositories
{
    public class BlogRepository : IBlogRepository
    {
        private BloggingContext _context;
        public BlogRepository(BloggingContext context)
        {
            _context = context;
        }

        ...

        public IList<Category> Categories()
        {
            return _context.Categories.OrderBy(c => c.Name).ToList();
        }
    }
}




Create a view model

We are going to display the complete sidebar through a single action.
We need a new single view model that wraps all the widget data.
Let's create a new view model called WidgetViewModel under the View Models folder.

namespace MyBlog.Models.View_Models
{
    public class WidgetViewModel
    {
        public WidgetViewModel(IBlogRepository blogRepository)
        {
            Categories = blogRepository.Categories();
        }

        public IList<Category> Categories { get; private set; }
    }
}


Currently our WidgetViewModel contains only property Categories of type IList<Category> , soon we are going to add other widgets data as well.

SidebarViewComponent

We are going to use View Components for the sidebar. View components are similar to partial views, but they are much more powerful.
A view component consists of two parts: the class (typically derived from ViewComponent) and the result it returns (typically a view).

Add a ViewComponent class

Create a ViewComponents folder and add the following SidebarViewComponent class: 

using Microsoft.AspNetCore.Mvc;
using MyBlog.Models.View_Models;
using MyBlog.Repositories;

namespace MyBlog.ViewComponents
{
    public class SidebarViewComponent : ViewComponent
    {
        private readonly IBlogRepository _blogRepository;

        public SidebarViewComponent(IBlogRepository blogRepository)
        {
            _blogRepository = blogRepository;
        }

        public IViewComponentResult Invoke()
        {
            var widgetViewModel = new WidgetViewModel(_blogRepository);

            return View(widgetViewModel);
        }
    }
}


Create the view component Razor view


Create the Views/Shared/Components folder. This folder must be named Components.
Create the Views/Shared/Components/Sidebar folder.
Create a Views/Shared/Components/Sidebar/Default.cshtml Razor view:


@model MyBlog.Models.View_Models.WidgetViewModel

<div id="sidebars">
    @Html.Partial("_Categories", Model.Categories)
</div>


Create the partial Views/Shared/_Categories.cshtml Razor view:

We are going to use cards to wrap the widget. 
@model IList<MyBlog.Models.Category>

<div class="card card-outline-secondary" style="background-color: #eceeef;">
    <div class="card-block">
    <h4 class="card-title">Categories</h4>

        @if (Model.Count > 0)
        {
    <ul class="list-group list-group-flush">
                @foreach (var category in Model)
                {
    <li class="list-group-item">
    <a asp-controller="Blog"
       asp-action="Category"
       asp-route-category="@category.UrlSlug"
       title="@category.Name">
                            @category.Name
                        </a>

                    </li>
                }
            </ul>
        }
        else
        {
    <p>No categories found!</p>
        }
    </div>
</div>


Update the List.cshtml to display the side bar. 
@model MyBlog.Models.View_Models.ListViewModel

<div id="content">
    <div class="row">
        <div class="col-md-8">
        <h1>@ViewBag.Title</h1>

            @Html.Partial("_Pager", Model)

            @if (Model.Posts.Count > 0)
            {
                foreach (var post in Model.Posts)
                {
                    @Html.Partial("_PostTemplate", post)

                }
            }
            else
            {
        <p>No posts found</p>
            }
            @Html.Partial("_Pager", Model)
        </div>


        <div class="col-md-4">
            <!-- Sidebar -->
            @await Component.InvokeAsync("Sidebar")
        </div>
    </div>
</div>
