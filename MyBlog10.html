Create the Contact Me page

In the Contact Me page we are going to use Cards and the Flexbox utilities.
We wrap the card in a div with class container. Then we wrap a div with class card-header an a div with class card-body in this card container. In the card-body div we wrap a form. We are going to manage the layout of this form with the Flexbox utilities.
To enable flex behaviors we have to add the class d-flex or one of the responsive variants (e.g. .d-sm-flex) to a container to create a flex container. The direct children elements of this flex container are transformed into flex items. This flex containers and items are able to be modified further with additional flex properties.
We will now change this form step by step by adding classes and looking at what the effect is.


Step 1

Without any class in the form, we see that the content is centered by the class text-center of the card component.


Step 2

By adding the class d-flex we create a flex container.
We add this class to the first four div elements of the form.
We also add the class mb-3 to add some more margin to the bottom of these flex containers.

Step 3

In extra small devices (<576px), we want the labels and associated controls to lay one below other. Starting from small devices and up to extra large devices, we want them aligned horizontal to each other. We can use the class flex-column first. This class will set a vertical direction to the flex items. Then we apply the flex-sm-row class. This class comes after the flex-column class, so this class overrides the previous class for small devices and up. Adding this class (flex-sm-row) will align the label and the input element horizontally. We want alo that the column with the labels in small devices an up take 3 gridcolumns and that the input elements take 8 gridcolumns. We have to add the classes col-sm-3 to the label element and col-sm-8 to the input element.

Step 4

On viewports small and wider we want to to align the text in the label vertically with the text of the placeholder in the input elements. Therefore we have to add the class align-items-center to the flexbox container. We don't apply these classes for the label containing the text Message. We also need to add the classes m-sm-0 and p-sm-0 to the first three labels to remove margin and padding. Otherwise, the text of these labels can not be vertically centered.

Step 5

We want the label to be left aligned for the extra small devices and right aligned on viewports sized SM (small) or wider. There we have to add the classes text-left and text-sm-right. We need to add the classes mr-sm-0 and pr-sm-0 to the label containing the text Message to remove the right margin and right padding for small devices and up.

Step 6

Add the class mr-sm-5 to all the labels of the form to create space between the labels and the input elements (in the message-label replace the class mr-sm-0 with class mr-sm-5 ).

Step 7

For extra small devices we want the button centered. On viewports small and wider we want the button right aligned. Therefore, we add the row, justify-content-center and justify-content-sm-end classes to the div element that contains the button. For medium and large devices we add some margin to the right with the classes mr-md-3 and mr-lg-5 to the button element.
